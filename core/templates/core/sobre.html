Sobre o Sistema de Relatórios

Sistema de Relatórios da Conservação Prefeitura

Este sistema foi desenvolvido para facilitar a criação e gestão de relatórios pelos funcionários da Conservação Prefeitura. A plataforma permite que usuários normais criem relatórios que são enviados para o banco de dados, enquanto administradores podem visualizar e gerenciar todos os relatórios do sistema.

Para Usuários Normais

  • Criar relatórios com título e conteúdo
  • Visualizar próprios relatórios
  • Acompanhar datas de criação
  • Sistema de autenticação seguro

Para Administradores

  • Visualizar todos os relatórios
  • Filtros e busca avançada
  • Identificar autor dos relatórios
  • Visualização detalhada

Funcionalidades do Sistema

  • Interface responsiva e moderna
  • Paginação automática
  • Mensagens de feedback
  • Sistema de permissões
  • Controle de acesso baseado em roles

Tecnologias Utilizadas

  • Django Framework
  • HTMX para interatividade
  • Bootstrap para interface
  • SQLite para banco de dados
  • Python 3.x
Como Usar o Sistema
Usuários Normais:
  1. Registrar-se no sistema
  2. Fazer login com suas credenciais
  3. Criar relatórios via formulário
  4. Visualizar relatórios próprios
Administradores:
  1. Fazer login como staff
  2. Acessar painel administrativo
  3. Visualizar todos os relatórios
  4. Utilizar filtros e busca
